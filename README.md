# Track-Your-Currency
Необходимо реализовать пет-проект 
для логирования изменений в полях любой модели. 
Нужно будет написать модель 
Пользователя ( необходимо написать кастомную с 
дополнительными полями ниже ) , 
модель History с помощь которой будет реализованы 
все изменения полей любой модели. 
Также для примера нужно будет взять модель Currency.

Суть проекта:
1) Чистая архитектура проекта с правильным испортом всех моделей
 и библиотек по PEP8
2) Логика по получению ежедневных данных по валютам
3) Использование Celery
4) Использование Celery crontab для выполнения запросов ежедневно
5) Обернуть все в Docker - docker-compose

Использованные архитектуры:
1) Python3.8
2) Django
3) djangorestframework
4) celery
5) Docker
6) docker-compose
